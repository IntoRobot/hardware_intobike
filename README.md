# IntoRobot IntoBike

IntoRobot IntoBike是IntoRobot为广大创客爱好者设计的一款开源平衡车，IntoBike使用IntoRobot Neutron Plus作为核心控制板，可以通过WiFi远程网络控制，可增加选配蓝牙模块通过蓝牙方式近场控制。
